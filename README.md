# RockPaperScissors
1. Computer chooses randomly rock, paper or Scissors
2. User can click the button rock, paper or scissors
3. The rules are as follows: rock wins over scissors, scissors wins over paper, and paper wins over rock. If both players choose the same option, it is a draw.
4. Added global variables to store store scores that shows the result
5. Added a quit button to quit the game.
